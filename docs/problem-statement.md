# Problem Statement

Engineering students in India spend weeks preparing for 
campus placements without knowing what specifically to fix. 
They get generic advice, no personalised gap analysis, and 
miss opportunities because they were simply not 
"placement-ready" in time.

## The Problem
There is no intelligent system that looks at a student's 
profile (resume, skills, CGPA, projects) and tells them:
- Exactly what skills are missing for their target role
- Which companies they currently match for
- What to do next — week by week
- How well they fit a specific job description

## Our Solution
An AI-powered multi-agent system built using Claude 3.5 
Sonnet and GPT-4o as LLM backbone, with Relevance AI as 
the orchestration platform.

The system has 4 specialized agents:
1. Resume Analyzer Agent
2. Web Search Agent  
3. JD Match Agent
4. Roadmap and Mock Interview Agent

## Target Users
Engineering students in India in their final or pre-final 
year preparing for campus placements.
