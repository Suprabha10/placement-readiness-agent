# 🎓 Campus Placement Readiness Agent

An AI-powered multi-agent system that analyzes a student's 
resume, identifies skill gaps, and generates a personalized 
placement preparation roadmap.

## 🤖 How It Works

The system has 4 specialized AI agents:

**Agent 1 — Resume Analyzer**
Extracts skills, projects, certifications and gives 
a profile strength score

**Agent 2 — Web Search Agent**
Searches the web in real time for current hiring trends, 
top companies, and salary ranges for the target role

**Agent 3 — JD Match Agent**
Paste any real job description and get a match score 
showing exactly what you have and what you are missing

**Agent 4 — Roadmap & Mock Interview Agent**
Generates a personalized 4-week preparation plan and 
5 mock interview questions based on your actual profile
